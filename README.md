## ΏθΟρ CUSTOMIZATION

### Change Terminal Colors
Edit `/etc/profile` and modify color codes:
```bash
export PS1='\[\e[38;5;196m\]S3RL\[\e[0m\] \[\e[38;5;51m\]$\[\e[0m\] '
```

### Add Custom Aliases
Edit `/home/ashov/.bashrc`:
```bash
alias rave='neofetch && figlet "RAVE ALL NIGHT"'
alias s3rl='echo "ΏθΝΙ S3RLINUX ΏθΤΑ - RAVE ALL NIGHT"'
```

### Modify MOTD
Edit `/etc/motd` and add your own ASCII art

### Recompile Kernel
```bash
cd /usr/src/linux
make menuconfig
make -j6
make modules_install
make install
```

## ΏθΦπ BUILDING FROM SOURCE

### Prerequisites
- Gentoo Linux system
- Ryzen 5000+ CPU (or compatible)
- 8GB+ RAM
- 100GB+ storage

### Build Steps
```bash
# 1. Install Gentoo normally (Stage 3)
# 2. Customize configs
# 3. Compile kernel
# 4. Install packages
# 5. Create Stage 4 tarball

tar --xattrs -cpf /tmp/s3rlinux-stage4.tar.xz -C / \
  --exclude=/proc \
  --exclude=/sys \
  --exclude=/dev \
  --exclude=/tmp \
  --exclude=/var/tmp \
  /
```
