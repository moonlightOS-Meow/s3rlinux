# Security Policy

| Version | Supported |
|---------|----------|

## Reporting a Vulnerability

If you find a security vulnerability, please open an issue. We take security seriously.

For urgent issues, DM @moonlightOS-Meow directly.

## Security Notes

- No sensitive data stored in repo
- Client-side only application
- No backend servers
- GitHub Pages deployment
